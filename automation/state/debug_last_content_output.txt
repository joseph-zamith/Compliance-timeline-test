    <!-- Section 1: UE et International -->
    <h2 style="color: #1c2837; font-size: 18px; font-weight: 700; margin: 0 0 20px 0; border-bottom: 3px solid #ff512c; padding-bottom: 10px;">UE et International</h2>

    <h3 style="color: #ff512c; font-size: 14px; font-weight: 700; margin: 0 0 12px 0;">ISO 13485:2016/A11 : enquête publique lancée (implication QMS majeure)</h3>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;">AFNOR annonce l'ouverture de l'enquête publique sur la révision A11 de la norme ISO 13485:2016. Les principaux changements pressentis intègrent des obligations de gestion IA/cybersécurité dans le QMS, traçabilité numérique renforcée, et gouvernance data. Cette révision fait suite aux recommandations post-MDR/AI Act. Publication prévue Q4 2026, avec une période de transition de 18 mois post-parution avant application obligatoire.</p>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;"><strong>Impact fabricant :</strong> audit QMS et mise à jour procédures dès Q1 2027. Deadline réponse enquête fin août 2026. Si logiciels IA utilisés ou prévus, participez à consultation.</p>
    <p style="color: #ff512c; font-size: 11px;"><a href="https://www.afnor.org/actualites/" style="color: #ff512c; text-decoration: none;">Source: AFNOR</a></p>

    <hr style="border: 0; border-top: 1px solid #e9ebee; margin: 20px 0;">

    <h3 style="color: #ff512c; font-size: 14px; font-weight: 700; margin: 0 0 12px 0;">MDCG 2025-6 : clarifications AI Act vs MDR/IVDR imminentes</h3>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;">La Commission européenne prépare une version révisée du guide MDCG 2025-6 (Annexe 1) traitant spécifiquement de l'interaction MDR/IVDR et AI Act. L'enjeu central : clarifier quand un SaMD contenant de la logique IA bascule en classification haut risque (AI Act Annexe III) versus simple conformité MDR. Les ambiguïtés actuelles figent les projets en attente de précision.</p>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;"><strong>Calendrier :</strong> publication fin juillet 2026 attendue. <strong>Action :</strong> revoir classification IA pour chaque SaMD. Algorithmes de diagnostic/prédiction en particulier peuvent basculer Annexe III (risques de biais, faux positifs critiques).</p>
    <p style="color: #ff512c; font-size: 11px;"><a href="https://health.ec.europa.eu/medical-devices-sector/new-regulations_en" style="color: #ff512c; text-decoration: none;">Source: EU Commission Health</a></p>

    <hr style="border: 0; border-top: 1px solid #e9ebee; margin: 20px 0;">

    <h3 style="color: #ff512c; font-size: 14px; font-weight: 700; margin: 0 0 12px 0;">ISO/TS 24971-2 : technical specification ML risk management finalisée</h3>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;">La TS 24971-2 sur la gestion des risques pour logiciels ML/IA est finalisée (après révision depuis enquête 2025). Spécification couvre : FMEA adapté ML (considération data/modèle), métriques performance (sensibilité/spécificité), validation data, monitoring post-lancement. Adoption progressive Q3/Q4 2026. Bien que non-obligatoire, elle devient de facto standard de référence pour organismes notifiés et audit conformité CE.</p>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;"><strong>Impact :</strong> projets ML/IA doivent aligner risk management sur 24971-2; audit QMS post-entrée vigueur inclura vérification.</p>
    <p style="color: #ff512c; font-size: 11px;"><a href="https://www.qualitiso.com/veille/" style="color: #ff512c; text-decoration: none;">Source: Qualitiso</a></p>

    <hr style="border: 0; border-top: 1px solid #e9ebee; margin: 20px 0;">

    <h3 style="color: #ff512c; font-size: 14px; font-weight: 700; margin: 0 0 12px 0;">MHRA AI Airlock Phase 2 : rapport publié, trajectoire UK 2027</h3>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;">L'agence britannique MHRA publie le rapport de Phase 2 du programme AI Airlock (lancé 2024 pour encadrement SaMD IA). Conclusions : introduction pré-soumission IA obligatoire à partir de 2027 pour SaMD haut risque (diagnostic, prédiction décision clinique). UK s'aligne de facto sur approche FDA pre-cert (US). Structure : dossier IA pré-soumission auprès MHRA, feedback avant dossier complet; calendrier soumission et approbation raccourci.</p>
    <p style="color: #1c2837; font-size: 12px; line-height: 1.6; margin: 0 0 12px 0;"><strong>Implication :</strong> fabricants avec ventes RU doivent intégrer pré-soumission MHRA 2027 dans roadmap réglementaire SaMD.</p>
    <p style="color: #ff512c; font-size: 11px;"><a href="https://www.dm-experts.fr/flash-reglementaire-normatif/" style="color: #ff512c; text-decoration: none;">Source: DM Experts / MHRA</a></p>
